# Demo 01: Personal Portfolio Landing Page
**Date:** 2026-03-30  
**Difficulty:** Beginner  
**Tech Stack:** React, SASS, Bootstrap 5

## 🎯 Learning Objectives
* Master SASS variables for theme consistency.
* Understand SASS nesting to keep CSS readable.
* Implement a responsive Bootstrap Navbar and Grid system.
* How to override `$primary` in Bootstrap before importing the library.
* Using SASS nesting to style sub-elements of the `.hero` class.

## 🛠️ Implementation Details
- **SASS Variables:** Defined `$primary-color`, `$secondary-color`, and `$font-main`.
- **Nesting:** Scoped all hero styles under `.hero-section { ... }`.
- **Bootstrap:** Used `container`, `row`, `col-md-6`, and `btn-primary`.
